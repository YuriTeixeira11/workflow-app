# Backend Lab

Projeto criado para estudo e prática de conceitos modernos de backend utilizando FastAPI.

## Tecnologias

- Python
- FastAPI
- SQLAlchemy
- SQLite
- Alembic
- JWT
- Docker
- Pytest

## Objetivo

Implementar gradualmente funcionalidades encontradas em aplicações reais:

- Autenticação JWT
- Refresh Token
- Upload de Arquivos
- Paginação
- Filtros
- Logs
- Cache
- Docker
- Testes Automatizados
